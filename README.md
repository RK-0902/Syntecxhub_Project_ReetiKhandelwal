# Syntecxhub Internship Projects

A collection of beginner-friendly Python projects demonstrating core programming concepts such as conditionals, loops, functions, file handling, and object-oriented programming.

## Projects

### 1. Calculator

A simple command-line calculator that performs basic arithmetic operations.

#### Features

* Addition
* Subtraction
* Multiplication
* Division

---

### 2. Number Guessing Game

A fun interactive game where the user tries to guess a randomly generated number.

#### Features

* Difficulty levels
* Higher/Lower hints
* Attempt counter
* Replay option

---

### 3. To-Do List Manager

A command-line task manager that helps users organize their daily tasks.

#### Features

* Add tasks
* View tasks
* Mark tasks as done
* Delete tasks
* Save tasks in a text file

---

### 4. Student Management System

A CLI-based application for managing student records using Object-Oriented Programming and JSON file storage.

#### Features

* Add student records
* Update student details
* Delete student records
* View all students
* Unique Student ID validation
* Data persistence using JSON
* Formatted console output

#### Student Information Stored

* Student ID
* Name
* Grade

#### Concepts Used

* Classes and Objects
* Constructors (`__init__`)
* Methods
* File Handling (JSON)
* Data Validation
* CRUD Operations
* OOP Principles

---

## Technologies Used

* Python 3

* File Handling
* Object-Oriented Programming (OOP)
